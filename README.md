# BarterManagementSystem

University project developed in Java for the management of a digital barter platform.

## Project Overview
The system allows users to exchange goods through a barter-based workflow, managing items, trade requests, and approvals.
The project was designed following Object-Oriented Programming principles and UML-driven design.

## Features
- User and item management
- Barter request creation
- Trade approval workflow
- Modular OOP architecture

## Technologies
- Java
- Object-Oriented Design
- UML (Class, Sequence, Use Case diagrams)

## Repository Structure
- `src/` → Java source code
- `docs/uml/` → UML diagrams
- `build/` → Executable JAR
- `archives/` → Original university submission archives

## How to Run
```bash
java -jar build/BarterManagementSystem.jar
